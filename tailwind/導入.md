npm経由でインストール
```
npm install -D tailwindcss
npx tailwindcss init
```
src/input.cssに３行記述
```
@tailwind base;
@tailwind components;
@tailwind utilities;
```
使われているユーティリティのみを使用するために常に下を起動
```
npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
```
main.tsx
```
import './input.css'
```
index.html
```
<link rel="stylesheet" href="/dist/output.css" />
```
