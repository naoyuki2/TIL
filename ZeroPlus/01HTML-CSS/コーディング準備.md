# Google Fonts 準備
https://fonts.google.com/

### index.htmlのhead内
```
    <link
      href="https://fonts.googleapis.com/css2?family=Noto+Sans+JP:wght@400;700&display=swap"
      rel="stylesheet"
    />
```

### style.cssのbody内

```
  font-family: "Noto Sans JP", sans-serif;
```

# bodyタグに共通するものを設定

### 例

```
body {
  color: #001c2b;
  font-family: "Noto Sans JP", sans-serif;
  font-size: 16px;
  line-height: 1.8;
  letter-spacing: 0.1em;
  background: #fafafa;
}
```
