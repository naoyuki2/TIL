# position: sticky;

要素を特定の位置に`粘着`させることができる。スクロールされても要素を固定したいときに使う。多分だけど、ヘッダーとかに使う。

下記では、要素がスクロールすると画面の上端に`粘着`する。`top:0`は必須。

```
セレクタ {
  position: sticky;
  top: 0;
}
```

# overflow

はみ出た部分をどのように扱うかを指定するやつ。

- `visible` : そのまま表示
- `hidden` : 隠れる
- `scroll` : 隠れてスクロールできるようになる

#   white-space

要素内のテキストの折り返しを制御する。

- `normal` : 自動で折り返される
- `nowrap` : 要素の幅を超えようとも自動で折り返さない
